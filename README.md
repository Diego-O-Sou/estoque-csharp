# Sistema de Controle de Estoque (C# + MySQL)

Aplicação desenvolvida para fins de estudo, com objetivo de praticar desenvolvimento backend em C# utilizando banco de dados MySQL. O sistema permite cadastrar, consultar, atualizar e excluir produtos de um estoque.

---

## Tecnologias Utilizadas
- C#
- .NET
- MySQL
- MySQL Connector / Entity Framework (se estiver usando)
- Visual Studio

---

## Funcionalidades
- Cadastro de produtos
- Consulta de produtos
- Atualização de informações
- Exclusão de itens
- Conexão com banco MySQL
- Interface simples (caso seja WinForms/WPF)
